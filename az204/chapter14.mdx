---
title: "Azure-da Hesablama (Compute) Həllərinin Hazırlanması"
---

Bu hissədə əvvəlcə bulud texnologiyalarının əsas anlayışlarını qısa şəkildə xatırladaraq Microsoft Azure-a giriş edirik və Azure üzrə bilməli olduğunuz əsas mövzuları izah edirik. Daha sonra Azure daxilində mövcud olan hesablama (compute) həlləri ilə tanış oluruq.

Burada ARM şablonlarının yaradılması və istifadəsi, Azure virtual maşınlarının qurulması və bu proses zamanı nəzərə alınmalı vacib məqamlar öyrədilir. Konteynerlərə giriş edilir və konteynerlərin və konteyner image-lərinin idarə olunması üçün istifadə olunan Azure xidmətləri izah olunur.

Hissənin davamında Azure App Service təqdim olunur. App Service üzərində web tətbiqlərin yaradılması, kodun yerləşdirilməsi, miqyaslama və konfiqurasiya imkanları sadə dildə açıqlanır. Bu hissənin son mövzusu Azure Functions-dur. Burada serverless funksiyaların yaradılması, inkişaf etdirilməsi və yayımlanması, həmçinin vəziyyət saxlayan durable functions anlayışı izah edilir.

Bu hissə AZ-204 imtahan suallarının təxminən **25–30%**-ni əhatə edir.

Bu bölməyə daxil olan fəsillər:

- **1-ci fəsil:** Azure və Bulud Əsasları _(Azure and Cloud Fundamentals)_
- **2-ci fəsil:** IaaS Həllərinin Tətbiqi _(Implementing IaaS Solutions)_
- **3-cü fəsil:** Azure App Service Web Tətbiqlərinin Yaradılması _(Creating Azure App Service Web Apps)_
- **4-cü fəsil:** Azure Functions-un Tətbiqi _(Implementing Azure Functions)_
